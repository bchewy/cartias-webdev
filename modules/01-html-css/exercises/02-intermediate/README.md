# Exercise 2: Styled Recipe Card

## Task
Create a recipe card with basic styling.

### Requirements
1. Create two files:
   - `recipe.html`
   - `style.css`
2. The recipe card should include:
   - Recipe title
   - Cooking time and servings
   - Ingredients list
   - Instructions list
   - A recipe image
3. Style requirements:
   - Center the title
   - Add padding around the content
   - Style lists differently for ingredients and instructions
   - Add a border around the card
   - Make the image responsive

### Hints
- Link your CSS file using `<link>`
- Use classes for styling
- Try different CSS properties like:
  - `padding`
  - `margin`
  - `border`
  - `text-align`
  - `max-width`

### Template
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Recipe Card</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <div class="recipe-card">
        <!-- Add your content here -->
    </div>
</body>
</html>
```

### Time: 15 minutes
